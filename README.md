# Web-Playground [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A collection of small web projects and experiments demonstrating HTML, CSS, and JavaScript concepts.

## Description

Web-Playground is a personal repository for experimenting with web technologies. Each subfolder contains a separate web project, with its own source files and structure. This repository is ideal for practicing web development, testing new ideas, and learning different web technologies.

## Projects Included

- **Fac Parody**: A fictitious parody website inspired by university news. This project is built using only HTML and CSS for training purposes.  
  - **Home Page**: Homepage displaying the latest fictitious "news" about university life.  
  - **Actuality Page**: Page detailing parody news with humorous articles.  
  - **About Page**: Page presenting the fictitious editors and photographers of the site with a humorous touch.  
  - **Subscription Page**: Page where users can "subscribe" to the fictitious newsletter.

*(More projects will be added over time as part of this repository.)*

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/Zynokou/Web-Playground.git
```

2. Navigate to the folder of the project you want to view, for example:

```bash
cd Web-Playground/fac-parody
```

3. Open the index.html file in your preferred web browser.

4. Explore the site or project as needed.

## Requirements

- Modern web browser (Chrome, Firefox, Edge, Safari, etc.)

- No server setup needed for static HTML/CSS projects
